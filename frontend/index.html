How it works · hybrid retrieval, rerank, citation-constrained answers

dense and BM25 retrieval fused with RRF because engineering acronyms defeat dense search alone; optional cross-encoder rerank; the generator is constrained to retrieved excerpts and every response carries document and page; retrieval and generation are graded separately because a fluent answer can still be ungrounded.

Notes · when it declines and why first run may wait
  1. This runs on a free tier that sleeps between visitors. First start takes 30 to 60 seconds; runs after that are quick.
  2. It answers only from the six documents on the shelf and is graded on declining everything else; a refusal on an off-shelf question is the system working.
  3. Graded questions come from the same 31-query set the published metrics use.
